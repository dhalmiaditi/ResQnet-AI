# ResQNet-AI

ResQNet is an AI-powered emergency response and disaster management platform designed to provide faster assistance during emergencies.

## Features
- AI-powered emergency assistance
- Women safety module
- Disaster management support
- Interactive web interface
- Location-based emergency response

## Tech Stack
- Python
- Flask
- HTML
- CSS
- JavaScript
- SQLite

## Installation

1. Clone the repository
```bash
git clone https://github.com/dhalmiaditi/ResQnet-AI.git
```

2. Install dependencies
```bash
pip install -r requirements.txt
```

3. Run the application
```bash
python app.py
```
